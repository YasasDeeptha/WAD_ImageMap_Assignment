Selected Destination

Navigation Steps

Page 1 - Faculty Entrance  (1.html)
Shows the main gate of the Faculty of Technology. The user has to
click on the gate area to go to the next step.

Page 2 - Main Entrance  (2.html)
Shows the area near the main entrance after going through the gate.
Click on the path at the bottom to continue walking forward.

Page 3 - Road to Academic Building  (3.html)
Shows the road going toward the main academic building. Click on the
road to go to the next step.

Page 4 - Main Academic Building  (4.html)
Shows the front of the main academic building. Click on the 2nd floor to go inside.

Page 5 - 2nd Floor Corridor  (5.html)
Shows the corridor on the 2nd floor. Click on the door to enter the
ICT Computer Laboratory.

Page 6 - ICT Computer Laboratory  (6.html)
This is the final destination page. It shows the ICT Computer
Laboratory.


How to open

Open index.html in a web browser. The first page will load automatically.


How to Run:
Open index.html in a web browser to start the navigation system.
